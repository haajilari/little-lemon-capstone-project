# Little Lemon Restaurant - Table Booking App

This project is the Capstone assignment for the Meta Front-End Developer Professional Certificate. It is a responsive web application that allows users to reserve a table at the Little Lemon restaurant.

## Features

- **Table Booking Form:** A fully functional form with controlled components.
- **Form Validation:** Client-side validation ensuring data integrity (HTML5 & JavaScript).
- **Unit Testing:** Comprehensive tests using React Testing Library to verify validation rules.
- **Accessibility (a11y):** Semantic HTML, ARIA labels, and keyboard navigation support.
- **UX Enhancements:**
  - Loading states provided visual feedback.
  - "Clear Form" functionality for user control.
  - Confirmation page upon successful submission.

## Technologies Used

- React.js
- JavaScript (ES6+)
- HTML5 & CSS3
- Jest & React Testing Library (for testing)
